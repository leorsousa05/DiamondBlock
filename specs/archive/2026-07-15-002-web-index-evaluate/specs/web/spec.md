# Web Index Evaluation Delta

## ADDED Requirements

### Requirement: Evaluate an indexed project

The Web UI SHALL allow a user to submit a semantic query with optional expected files and symbols and SHALL display top-1, top-3, and top-5 hit results.

### Requirement: Explain token savings

The Web UI SHALL display retrieved tokens, baseline tokens, and reduction percentage, labeled as approximate.

### Requirement: Show parser coverage

The Web UI SHALL display AST, simplified, and fallback chunk counts and SHALL expose each chunk's parser mode and relation count when available.

## MODIFIED Requirements

The Codebase Index page SHALL contain three tabs: Status & Run, Chunks, and Evaluate.

## REMOVED Requirements

None.
